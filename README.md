# code patch generate

1.  download code_patch.sh
2.  open a git bash
3.  run code_patch.sh as follow  

    code_patch.sh [PATCH_NAME] [PATCH_SHA_1] [PATCH_REPO]  

    eg.  
    ```bash
    code_patch.sh test 0eb3337b847d080590b07c05f98138592aa7f49e 'c:\git'
    ```
4.  [PATCH_NAME].tgz will be generated
